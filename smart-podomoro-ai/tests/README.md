# 🍅 Smart Pomodoro AI

Aplikasi **Pomodoro Timer** cerdas yang tidak hanya menghitung waktu, tapi juga "belajar" memahami tingkat fokus penggunanya menggunakan **Artificial Intelligence (PyTorch)**.

## 🚀 Fitur Unggulan
- **AI Focus Analyzer:** Menggunakan *Neural Network* sederhana (PyTorch) untuk memberikan saran berdasarkan pola kerja dan jumlah interupsi (Stop/Pause).
- **Dynamic Visuals:** Animasi "Lingkaran Obat Nyamuk" yang berkurang seiring berjalannya waktu.
- **Gamification:** Sistem Level Fokus (⭐) untuk memotivasi pengguna menyelesaikan sesi.
- **Deep Work Mode:** Tema warna gelap (Dark Mode) untuk mengurangi kelelahan mata.

## 📂 Struktur Direktori
```text
smart-pomodoro-ai/
├── data/               # Penyimpanan model AI (.pth)
├── src/                # Kode sumber utama
│   ├── main.py         # Titik masuk aplikasi
│   ├── ai_logic.py     # Logika Neural Network (PyTorch)
│   └── gui.py          # Antarmuka Tkinter
├── requirements.txt    # Daftar dependensi
└── README.md           # Dokumentasi ini
