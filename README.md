# 🐍 My Python Learning Journey

> *"It's never too late to start again."*

Welcome to my personal repository where I document my journey of learning Python from scratch — or rather, re-learning it after a long pause.

## 📖 Background

I used to work as a Senior Software Engineer at a retail company in Indonesia. Due to financial constraints, I made the difficult decision to resign from the IT world and try my luck in the manufacturing industry.

For years, I struggled with limitations — both in resources and opportunities. Meanwhile, technology kept advancing faster than I realized.

Now, at **30 years old**, I've decided to pick up where I left off. This repository is my comeback.

## 🎯 Purpose

- Rebuild my programming fundamentals with Python
- Document daily or weekly progress
- Stay consistent despite a full-time job in manufacturing
- Prove that learning has no age limit
